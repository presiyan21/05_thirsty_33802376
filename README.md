# Thirsty Student

A small Node.js web application for *The Thirsty Student* drinks shop.  
This project is a simple multi-page website that demonstrates how to use **Express**, **EJS templates**, and **middleware** to handle **GET** and **POST** requests, collect user data through forms, and serve styled web pages.

It features pages for searching products, registering users, and submitting customer surveys.  
In addition, it includes form validation, external CSS styling, and an expandable shop list near **Goldsmiths University of London**.
